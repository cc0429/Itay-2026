# Italy Travel 2026 Web App

这是 iPhone 可“添加到主屏幕”的 Web App 版本。

## 使用方法

1. 解压本 zip。
2. 将整个文件夹上传到 Netlify / Vercel / GitHub Pages。
3. 用 iPhone Safari 打开部署后的 HTTPS 链接。
4. 点击 Safari 底部分享按钮 → 添加到主屏幕。
5. 之后从主屏幕图标打开即可像 App 一样使用。

注意：iPhone 的 Web App/PWA 需要通过 Safari 打开网页链接安装；不能直接把 zip 或本地 html 当作 iOS App 安装。地图瓦片和 Google Maps 路线需要网络。首次联网打开后，行程页本身会被缓存。
